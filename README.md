# MVP THA

### Requisitos

node >= v16.15.0
yarn 1.22.18

### Instalação

`yarn`

### Utilização

Rodar a aplicação

`yarn run start`

Rodar os testes de unidade

`yarn run test:unit`

Rodar os testes de integração

`yarn run test:integration`

Rodar os testes end 2 end

`yarn run test:e2e`
